# Skill Assessment Taxonomy

## Definitions

- **Function**: A broad organizational domain representing a major area of work (e.g., Product, Engineering, Marketing, Data).
- **Role**: A specific job within a function, defined by responsibilities and expected contributions (e.g., Product Manager, Backend Engineer).
- **Subject**: A focused topic or domain within a role that can be taught and assessed (e.g., PRDs, API Development).
- **Outcome**: A clear statement of what a user should know, understand, or be able to do. It is measurable and observable.
- **Skill**: The ability to perform a task effectively, developed through practice and experience.
